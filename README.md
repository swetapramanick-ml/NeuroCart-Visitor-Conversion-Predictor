# NeuroCart Visitor Conversion Predictor 🛒

📌 Project Overview
This project predicts whether a website visitor will make a purchase based on browsing behavior using Machine Learning.

- Data preprocessing & cleaning
- Feature engineering
- Classification models (Decision Tree, etc.)
- Model evaluation (Accuracy, Precision, Recall, F1-score)

- Scikit-learn
- Matplotlib / Seaborn
